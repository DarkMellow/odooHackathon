# Gemini Agent Instructions

Please follow these simple and straightforward rules when working on this project:

1. **No Self-Testing**: Do not run any tests by yourself to verify features and implementations. The user will verify the features and provide reviews/feedback.
2. **Do Not Run Dev/Preview Scripts**: Never run any development servers (e.g., `pnpm dev`, `npm run dev`) or preview scripts that may interfere with or affect the user's active workflow.
3. **Follow Documentation**: Strictly follow the specifications and guidelines laid out in the markdown files located in the `docs/` directory.
4. **Use Available Skills**: Always prioritize using the configured agent skills for executing specialized tasks.
5. **Use Relative Paths**: Always use relative paths when referencing documents, markdown files, and assets (e.g., `docs/PRD.md`, `docs/TRD.md`).
